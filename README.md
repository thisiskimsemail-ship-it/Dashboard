# Wade Master Dashboard

Private internal dashboard for the Wade Marketing team. Refreshed weekly from Supermetrics (GA4, Google Ads, LinkedIn, HubSpot).

## Access

This site is gated by a client-side password. Ask Kim Howells for access.

## Refresh cadence

A new version of `index.html` is generated every Monday by Claude using the [Wade Insights Agent](../Wade%20Insights%20Agent/SKILL.md). To update the live site, replace `index.html` in this repo with the latest dated copy from OneDrive (`_CLAUDE - Wade Data and Insights/3. Outputs/Dashboards/Wade_Master_Dashboard_YYYY-MM-DD.html`), commit, and push.

## Tabs

Tabs follow the marketing funnel:

- **Overview** · **Performance** · **Full Funnel** — summary
- **Organic Content** — LinkedIn awareness (top of funnel)
- **Community Hub** · **Website Engagement** — content engagement (mid funnel)
- **The Studio** — active tool users (mid–lower funnel)
- **EDM Engagement** — nurture
- **Program Funnels** · **Audience Funnels** — conversion (bottom of funnel)
- **Definitions** · **Things to know** — meta
